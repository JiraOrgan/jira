## 공통 패턴

- **내비게이션**: 프로젝트 컨텍스트는 URL `{key}`와 전역 스토어 이중 유지.  
- **권한**: 버튼·메뉴는 RBAC에 따라 숨김/비활성(FR-030).  
- **에러**: 409 `WORKFLOW_VIOLATION`, `WIP_LIMIT_EXCEEDED` 토스트 + 문구.
