## 3. 액션 아이템

1. Phase 3 착수 시 `build.gradle`/Security 설정에 **BCrypt strength ≥ 12** 명시.  
2. 성능 NFR은 **ERD 인덱스·JQL SPIKE** 산출물과 RTM 연결.  
3. 운영 전 **체크리스트**: TLS 종단, WAF, 백업 RPO/RTO 문서화.
